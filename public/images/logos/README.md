# Client / partner logos

Drop official partner logo files here to have them render automatically in the
**Clients / Partnerships** section. No code changes are needed.

Naming convention (by slug, matching `lib/data.ts`):

```
reliance.svg   (or .png)
hcl.svg
ibm.svg
crif.svg
adp.svg
bayer.svg
```

Resolution order per logo: `<slug>.svg` → `<slug>.png` → styled text wordmark fallback.

Use logo files you are licensed to use (official brand/press-kit assets). These
trademarked logos are intentionally **not** bundled with this demo repository.
